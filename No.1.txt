ข้อที่ 1
มีคุณสมบัติ ค่าไม่ซ้ำกัน การกระจายค่า การคำนวฯง่าย
ค่าไม่ซ้ำกัน : Hash function ควรมีคุณสัมบัติไม่ซ้ำกัน ข้อมูลที่เเตกต่างกันจะต้องให้ค่า Hash ที่เเตกตต่างกันด้วย
การกระจ่ายค่า : Hash function ควรกระจายค่า Hash ให้เเตกต่างกันอย่างมากที่สุด เมื่อมีข้อมูลที่มีลักษณะที่คล้ายกันอยู่ใน เวกเตอร์ของข้อมูลห้ Hash ค่าของข้อมูลเหล่านั้นกระจายอย่างเท่าทันที สามมารถช่วยลดความน่าจะเป็นที่ข้อมูล
การคำนวณง่าย : Hash function มีความเร็วในการคำนวณเพื่อให้งานได้อย่างมีประสิทธิภาพ ซึ่งเป็นสิ่งสำคัญนั้นเป็นไปได้
